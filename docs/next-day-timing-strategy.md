# 明日(T+1) 短線買賣時機策略 — 設計與計算文件

> 對應實作：[`backend/daily_signal.py`](../backend/daily_signal.py)，合併於 API
> `GET /api/portfolio/position-analysis` 的 `daily` 區塊，呈現在「資產管理 →
> 分析」與「K 線圖 → 明日買賣時機」面板。
>
> 與 [倉位管理策略](./position-management-strategy.md) 互補：
> - 倉位管理回答「持有後該減/加/出」；
> - 本文回答「**明日(T+1) 在什麼價位、什麼條件下進出最合適**」。
>
> **術語**：文中「明日 / T+1」一律指「以今日收盤為基準的下一個交易日」，
> 即第二個交易日（今日為第一日），**不是第三日**。本文為技術分析輔助，不構成投資建議。

---

## 1. 資料與指標

需要日線 OHLCV（≥20 根，否則回傳 `INSUFFICIENT_DATA`）。計算下列最新值：

| 指標 | 公式 / 說明 |
|---|---|
| MA5 / MA10 / MA20 / MA60 | 收盤價移動平均；另取 MA5、MA10 前一日值判斷上彎/倒掛 |
| 量能均線 | `vol_ma5 = volume.rolling(5).mean()`、`vol_ma10 = rolling(10)` |
| 量比 | `vr5 = 量 / vol_ma5`、`vr10 = 量 / vol_ma10` |
| 放量 / 縮量 | 放量 = `vr5 ≥ 1.2`；縮量 = `vr5 < 0.8` |
| 前高 | 近 60 日（不含今日）最高價 = 上方壓力參考 |
| 平台支撐 | 近 20 日（不含今日）最低價 |
| 上影線比 | `(high − max(open,close)) / (high − low)` |
| 下影線比 | `(min(open,close) − low) / (high − low)` |
| 實體比 | `|close − open| / (high − low)` |
| 日漲幅 | `close / 前收 − 1` |

---

## 2. 位置判斷

依序判斷（先命中先返回）：

```
near_prev_high = 前高存在 且 close ≥ 前高 × 0.98
高位      : close / MA20 ≥ 1.15
前高附近  : near_prev_high 且 未實際突破前高
低位      : close ≤ MA20 × 0.98 且 close ≤ MA60
均線附近  : |close/MA5 − 1| < 0.02 或 |close/MA10 − 1| < 0.02
平台/中段 : 其餘
```

---

## 3. K 線型態分類

取第一個命中：

1. **高位衝高回落** — 位置在高位/前高附近，且上影線比 ≥ 0.30，且放量
2. **放量大陽線（帶下影）** — 收漲、實體比 ≥ 0.5、放量；下影線比 ≥ 0.2 則為「帶下影」
3. **放量突破前高/平台** — 收盤突破前高且放量
4. **長上影**（上影線比 ≥ 0.35）— 收漲為「紅K長上影」，否則「綠K長上影」
5. **長下影**（下影線比 ≥ 0.35 且實體比 < 0.5）— 低位為「低位長下影」，否則「長下影小實體」
6. **上下影均衡**（|上影比 − 下影比| < 0.12 且實體比 < 0.4）— 收漲/收跌
7. 否則 **小陽線 / 小陰線**

## 4. 短期均線倒掛（MA5 < MA10）★ 新增

短線動能的關鍵結構（如今日的 TSMC）：

```
ma_inverted    = MA5 < MA10                       # 短期均線倒掛（短均死叉）
ma_cross_today = ma_inverted 且 前一日 MA5 ≥ MA10  # 今日剛形成死叉，訊號更強
```

影響：
- 趨勢分 **−2**（倒掛）；若為**死叉當天**再 **−1**；
- 偏空觸發新增「反彈不過 MA10 → 短均倒掛壓制」；
- 輸出結構提示 `note`：反彈不過 MA10 不宜追多，等放量站回 MA10 之上再轉強。

## 5. §13 系統化打分

四項分數加總 → 總分 → 偏向。

### 5.1 趨勢分
| 條件 | 分數 |
|---|---:|
| close > MA5 | +1（否則 −2） |
| close > MA10 | +1（否則 −3） |
| MA5 上彎（> 前一日 MA5） | +1 |
| MA10 上彎 | +1 |
| MA5 > MA10 > MA20（多頭排列） | +2 |
| **MA5 < MA10（短均倒掛）** | **−2** |
| **死叉當天（剛倒掛）** | **−1** |

### 5.2 K 線分
| 條件 | 分數 |
|---|---:|
| 放量大陽線 | +3 |
| 放量大陽線帶下影 | +2 |
| 低位長下影 | +2 |
| 紅K長上影（高位/前高附近） | −2 |
| 綠K長上影 | −3 |
| 高位衝高回落 | −3 |
| 上下影均衡收跌 | −1 |
| 跌破前一日低點 | −3 |

### 5.3 量能分
| 條件 | 分數 |
|---|---:|
| 量比 vr5 > 1.0 | +1 |
| 量比 vr10 > 1.0 | +1 |
| 放量突破前高 | +3 |
| 放量長上影 | −2 |
| 放量收跌（非收漲且放量） | −3 |
| 縮量回踩不破平台 | +1 |

### 5.4 位置分
| 條件 | 分數 |
|---|---:|
| 低位 + 長下影 | +2 |
| 放量突破前高/平台 | +3 |
| 平台/均線附近 + 縮量不破支撐 | +2 |
| 接近前高但未突破 | −1 |
| 高位衝高回落 | −3 |
| 放量突破前高且上影短（站穩） | +3 |

### 5.5 偏向解讀
| 總分 | 偏向 |
|---|---|
| ≥ 7 | 偏多（可找買點） |
| 4 ~ 6 | 偏多觀察（等確認） |
| 1 ~ 3 | 中性（不急操作） |
| −1 ~ 0 | 偏弱（降低期待） |
| ≤ −2 | 風險較高（不宜追高/應減倉） |

## 6. 明日(T+1) 條件式觸發

以今日的高/低/均線/前高為基準，設定明日的條件反應（不預測漲跌）。

**偏多觸發（情況逐條）**
- 站上今日高點 → 偏多延續（漲勢可能續走）
- 回踩 MA5 不破再轉強 → 可低吸（回落到均線不破時逢低買進）
- 放量站上前高並收穩 → 突破確認（過前高、可視為轉強）

**偏空觸發（情況逐條）**
- 跌破今日低點 → 承接失效（下方買盤頂不住）
- 跌破 MA5 → 超短線轉弱（不宜追）
- 跌破 MA10 → 短線趨勢轉弱（該減倉或退出）
- （倒掛時）反彈不過 MA10 → 短均倒掛壓制
- 反彈不過前高 → 上方壓力持續（衝不過、賣壓重）

---

## 7. 建議買法

| 情境 | 買法 |
|---|---|
| 放量突破前高/平台 | **突破買**（突破壓力位時買進）：放量站上前高收穩、明日不跌回突破位 |
| 站在 MA5/MA10 之上且總分 ≥ 1 | **回踩買**（拉回到均線不破再買，即低吸）：回踩 MA5/MA10 不破、出現承接再進 |
| 長上影/長下影/衝高回落/分歧 K | **確認買**（等明天走強確認再買）：明日收紅站回今日高點、量能溫和放大再進 |
| 其餘 | 訊號分歧（多空不明），等明日確認再決定 |

## 9. 與倉位管理如何配合

| 面向 | 來源 | 回答 |
|---|---|---|
| 倉位面 | [position_strategy](./position-management-strategy.md) | 該減/加/出/續抱（狀態 + 動作 + 停損/目標） |
| 時機面 | daily_signal（本文） | 明日在什麼價位、什麼條件進出 |

範例：倉位面「轉弱 → 分批減倉」+ 時機面「反彈不過前高 X 偏空、跌破今日低 Y 加速、MA5<MA10 倒掛壓制」→ 明日反彈到 X 不過就減、跌破 Y 就執行。

## 10. 風險控制（沿用框架原則）

- 不靠單根 K 線決策，要看位置、均線方向、量能、前高壓力、支撐是否有效、明日是否確認。
- 任何買入理由都先定**失效點**（如回踩 MA5 不破 → 跌破 MA5 即失效）。
- 高位不追長上影；放量不等於永遠偏多（可能是出貨/換手）。
- 隔日交易核心不是預測，而是條件反應：**突破哪裡才買、跌破哪裡就賣、震盪在哪裡就等。**
